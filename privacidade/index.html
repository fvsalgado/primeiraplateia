Política de Privacidade

Última actualização: Março 2026 · Versão 1.0

Resumo simples: não recolhemos dados pessoais. Não usamos cookies de rastreamento. Não vendemos nada a ninguém. Esta página é estática — o teu browser faz os pedidos, nós não sabemos quem és.

Quem somos

A Primeira Plateia (primeiraplateia.pt) é um projecto independente de agenda cultural, desenvolvido por Fábio. Contacto: [email protected].

Que dados recolhemos

Este site é completamente estático. Não tem servidor próprio, não tem base de dados, não tem sistema de autenticação. Não existe qualquer mecanismo de registo, login ou formulário que recolha dados pessoais.

Os únicos dados que o site armazena localmente no teu dispositivo são:

  • Preferência de tema (modo claro/escuro) — guardada no localStorage do teu browser, apenas no teu dispositivo, nunca enviada para qualquer servidor.

Cookies

A Primeira Plateia não usa cookies de qualquer tipo — nem de sessão, nem de rastreamento, nem de publicidade. O localStorage usado para a preferência de tema não é um cookie e não é partilhado com terceiros.

Hosting e dados de acesso

O site está alojado no GitHub Pages. O GitHub pode registar endereços IP e dados básicos de acesso nos seus próprios logs de servidor, de acordo com a sua política de privacidade (docs.github.com). A Primeira Plateia não tem acesso a esses logs.

Fontes de conteúdo externas

O site carrega conteúdo de fontes externas que podem registar pedidos:

  • Google Fonts — tipografia (Playfair Display e Inter). Ao carregar a página, o teu browser faz um pedido aos servidores da Google para obter as fontes.
  • OpenStreetMap — mapas nas páginas de teatro, carregados via iframe com loading="lazy".
  • Imagens dos teatros — carregadas directamente dos sites dos teatros originais.

Recomendamos que consultes as políticas de privacidade de cada um destes serviços.

Dados de espectáculos

A programação exibida é recolhida automaticamente dos sites públicos dos teatros. Não recolhemos nem publicamos dados pessoais de público. Os dados de membros de equipa artística (encenadores, actores, etc.) são informação de carácter público divulgada pelos próprios teatros.

Os teus direitos (RGPD)

Como não recolhemos dados pessoais identificáveis, não há dados sobre ti para consultar, corrigir ou eliminar. Se tiveres dúvidas sobre dados de terceiros (ex: imagens, nomes em fichas técnicas), contacta directamente o teatro em questão.

Esta política pode ser actualizada. A data de "última actualização" no topo reflecte sempre a versão em vigor.
